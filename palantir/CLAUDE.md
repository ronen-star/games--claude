# CLAUDE.md

This file provides guidance to Claude Code when working in the `palantir/` project.

## Project Overview

**Palantir FY2025 Analysis** — business intelligence and investor presentation project covering Palantir Technologies. Contains financial analysis, KPI dashboards, competitive mapping, and a rendered video presentation.

## Structure

```
palantir/
├── palantir-video/        Remotion video project (React + TypeScript)
├── palantir_analysis/     Data files, dashboards, KPI reports
├── *.flowchart.excalidraw Diagrams: growth engine, competitive map, KPI dashboard
└── תמונה-*.png            Screenshot exports of diagrams
```

## palantir-video (Remotion)

Built with [Remotion](https://www.remotion.dev/) — video creation in React.

```bash
npm start          # preview in browser
npm run render     # render to out/palantir.mp4
npm run render:gif # render to out/palantir.gif
```

Run commands from inside `palantir-video/`.

## palantir_analysis

Data and reports generated by the business intelligence pipeline:
- `palantir_financials.csv` / `palantir_kpis.json` — source data
- `palantir_dashboard.html` — interactive KPI dashboard
- `palantir_kpi_report.json` / `palantir_metric_validation.json` — generated reports
- `palantir_competitors.csv` — competitive landscape data
