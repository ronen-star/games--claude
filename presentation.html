// 01 — ארכיטקטורה

הרעיון הפשוט

  • ▸ קובץ יחיד — index.html מכיל את כל הקוד. אין ספריות, אין npm, אין build.
  • ▸ Canvas פנימי ברזולוציה של 480 × 640 פיקסלים, מוגדל ב-CSS להתאים לכל חלון.
  • ▸ requestAnimationFrame מריץ את לולאת המשחק — חלק, 60fps, נייטיב לדפדפן.
  • ▸ מכונת מצבים מנהלת את כל זרימת המשחק — כל מצב מקבל זוג פונקציות משלו.
Vanilla JS HTML5 Canvas Web Audio API zero dependencies single file

// 03 — מנוע פיזיקה

כדור ופגיעות

  • ▸ הכדור הוא אובייקט פשוט: { x, y, vx, vy, spd }
  • ▸ פגיעה בקיר: ציר המהירות מתהפך. כל פגיעה = +1 נקודה
  • ▸ פגיעה במחבט: זווית הקפיצה נקבעת לפי המיקום על המחבט — עד ±70°
  • ▸ כל 5 פגיעות במחבט — מהירות ×1.1, מקסימום 15
  • ▸ הכדור מסתובב ויזואלית — זווית += vx / רדיוס בכל פריים

רינדור ו-HUD

  • ▸ שכבת סקאנליינים — רצועות שחורות שקופות כל 4 פיקסלים, אפקט CRT רטרו.

// 06 — מערכת קלט

שליטה במחבט

מקלדת
  • ▸חצים ← → או A / D
  • ▸Space / Enter להתחלה
עכבר
  • ▸המחבט עוקב אחרי מיקום ה-X של הסמן
  • ▸תנועה חלקה עם Lerp: pad.x += (mouseX − pad.x) × 0.2
מגע (מובייל)
  • ▸touchmove שולט במחבט
  • ▸touchstart מתחיל משחק
  • ▸מתואם לקואורדינטות ה-Canvas
אתחול אודיו
  • ▸AudioContext נוצר רק בפעולה הראשונה — עוקף מדיניות Autoplay של הדפדפן
